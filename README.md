## SEO

| File | Purpose |
|---|---|
| `robots.txt` | Instructs crawlers вЂ” allows all, blocks `/admin/` and `/private/` |
| `sitemap.xml` | Submitted to Google Search Console for indexing |
| JSON-LD (in `<head>`) | Structured data: `GeneralContractor` schema with address, geo, contacts, logo |
| Canonical tag | `https://mth-mv.com/` вЂ” prevents duplicate content issues |
| Open Graph tags | Controls preview appearance on LinkedIn, Facebook, WhatsApp |
